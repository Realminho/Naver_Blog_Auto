# PC버전 사용법

이 버전은 Windows PC에서 실행하는 버전입니다.
AI로 글을 만들고, 네이버 블로그처럼 미리보기한 다음, 마음에 들면 Chrome을 열어 네이버 블로그 글쓰기 화면에 제목과 본문을 자동 입력합니다.

중요: 네이버 로그인과 최종 발행 버튼은 직접 해야 합니다. 자동 발행은 실수로 글이 올라갈 수 있어서 넣지 않았습니다.

---

## 1. 압축 풀기

압축파일을 바탕화면이나 다운로드 폴더에 풀어주세요.

폴더 안에 `PC버전` 폴더가 있습니다.

---

## 2. API 키 넣기

`PC버전` 폴더 안의 `.env` 파일을 메모장으로 엽니다.

아래 줄을 찾습니다.

```txt
NVIDIA_API_KEY=YOUR_API_KEY
```

`YOUR_API_KEY`만 본인의 NVIDIA API 키로 바꿉니다.

예시:

```txt
NVIDIA_API_KEY=nvapi-xxxxxxxxxxxxxxxx
```

---

## 3. base_url 위치

`.env` 파일은 아래처럼 되어 있어야 합니다.

```txt
NVIDIA_BASE_URL=https://integrate.api.nvidia.com/v1
```

이 주소는 브라우저 주소창에 직접 들어가는 웹페이지가 아니라, 프로그램이 내부에서 `POST /chat/completions` 요청을 보낼 때 쓰는 API 기본 주소입니다.
따라서 주소창에 `https://integrate.api.nvidia.com/v1`만 직접 입력하면 404가 뜨는 것이 정상일 수 있습니다.
실제 호출 주소는 `https://integrate.api.nvidia.com/v1/chat/completions`입니다.

---

## 4. PC에서 실행하기

`실행_PC용.bat` 파일을 더블클릭합니다.

처음 실행할 때는 필요한 패키지를 설치하기 때문에 시간이 조금 걸릴 수 있습니다.
브라우저가 자동으로 열리면 프로그램을 사용하면 됩니다.

브라우저가 자동으로 안 열리면 주소창에 아래 주소를 입력하세요.

```txt
http://localhost:8501
```


---

## 4-1. 새로고침해도 입력값 유지

입력칸 값은 자동으로 임시저장됩니다.
브라우저를 새로고침해도 제품명, 주소, 특징, 가격, 말투 같은 입력값과 최근 생성 결과가 최대한 복원됩니다.

단, 사진/동영상 파일 자체는 브라우저 보안 정책 때문에 새로고침 후 다시 선택해야 할 수 있습니다.
입력값을 완전히 지우고 싶으면 화면의 `입력값 임시저장 초기화` 버튼을 누르세요.

---

## 5. 사용 흐름

1. 제품명/장소명, 특징, 주소, 가격, 주차, 영업시간 등 아는 정보만 입력합니다.
2. 모르는 칸은 비워둬도 됩니다.
3. 사진과 동영상을 업로드합니다.
4. 사진 설명, 동영상 설명을 적습니다.
5. `생성하기`를 누릅니다.
6. `네이버식 미리보기` 탭에서 실제 블로그처럼 보이는지 확인합니다.
7. `수정/작성하기` 탭에서 제목과 본문을 수정합니다.
8. 마음에 들면 `미리보기 내용으로 네이버 블로그에 작성하기`를 누릅니다.
9. Chrome이 열리면 네이버 로그인은 직접 합니다.
10. 프로그램이 제목과 본문을 네이버 글쓰기 화면에 넣습니다.
11. 사진/동영상은 자동 업로드 시도를 켤 수 있지만, 실패할 수 있으므로 최종 확인하세요.
12. 네이버 화면에서 직접 확인 후 발행합니다.

---

## 6. 네이버 자동 입력이 안 될 때

네이버 편집기 구조가 바뀌면 자동 입력이 실패할 수 있습니다.
그럴 때는 프로그램 안의 `제목 복사`, `본문 복사` 버튼을 사용해서 직접 붙여넣으면 됩니다.

---

## 7. EXE 만들기

`build_exe.bat`을 더블클릭하세요.

완성 위치는 아래입니다.

```txt
dist\NaverBlogWriter\NaverBlogWriter.exe
```

`dist\NaverBlogWriter` 폴더 전체를 보관하거나 다른 PC로 옮기면 됩니다.

---

## 8. 같은 와이파이에서 핸드폰으로 접속하기

PC와 핸드폰이 같은 와이파이에 연결되어 있다면 `실행_핸드폰_같은와이파이용.bat`을 더블클릭하세요.

검은 창에 아래처럼 주소가 나옵니다.

```txt
핸드폰 접속 주소: http://192.168.0.12:8501
```

핸드폰 Chrome/Safari 주소창에 그 주소를 입력하면 됩니다.

이 방식은 PC를 켜두고 있어야 합니다.


## 생성하기 후 미리보기가 안 보일 때

이번 버전부터 NVIDIA API가 401/404/429 오류를 내더라도 화면이 비어 있지 않게, 자동으로 템플릿 초안을 만들어 미리보기를 보여줍니다.

확인할 점:

1. `.env`에는 `Bearer`를 쓰지 말고 `nvapi-...` 키만 넣으세요.
2. `NVIDIA_BASE_URL=https://integrate.api.nvidia.com/v1` 로 두세요.
3. `https://integrate.api.nvidia.com/v1`을 브라우저로 열면 404가 뜰 수 있습니다. 정상입니다. 프로그램은 실제로 `/chat/completions` 주소로 POST 요청을 보냅니다.
4. API 키가 틀리거나 노출되어 폐기된 경우에는 401이 뜹니다. NVIDIA에서 새 키를 발급받아 넣으세요.


## NVIDIA API 타임아웃이 날 때

`Read timed out` 오류는 API 키 오류가 아니라 NVIDIA 서버 응답이 오래 걸려서 끊긴 상태입니다.
이번 버전은 기본 `NVIDIA_MAX_TOKENS=2500`, `NVIDIA_TIMEOUT_SECONDS=240`으로 조정했습니다.
그래도 느리면 왼쪽 설정에서 **AI 응답 길이 제한**을 1500~2500으로 낮춘 뒤 다시 생성하세요.

## 참고 링크로 글을 더 풍부하게 만드는 방법

프로그램의 `참고 링크` 영역에 아래 링크를 여러 개 넣을 수 있습니다.

```txt
리뷰할 제품/장소 정보 링크 여러 개
- 공식 홈페이지
- 스마트스토어/판매 페이지
- 네이버지도 링크
- 인스타그램/SNS 링크

참고할 다른 블로그/리뷰 링크 여러 개
- 다른 리뷰어가 작성한 네이버 블로그 후기
- 방문 후기/사용 후기 페이지
```

`생성하기`를 누르면 프로그램이 공개 페이지의 제목, 설명, 본문 일부를 읽어서 AI에게 참고자료로 전달합니다. 다른 리뷰어 글은 그대로 복사하지 않고, 분위기·장단점·방문 팁을 참고해서 새 문장으로 재구성하도록 설정했습니다.

주의: 로그인이 필요한 페이지, 차단된 페이지, 너무 무거운 페이지는 내용을 읽지 못할 수 있습니다. 그래도 입력한 링크 주소 자체는 참고자료로 남고, 읽힌 링크만 글 생성에 반영됩니다.


---

## 쿠팡 제품 링크 보완

이번 버전은 쿠팡 제품 링크를 `리뷰할 제품/장소 정보 링크 여러 개` 칸에 넣었을 때, 공개 HTML에서 읽히는 상품명, 가격, 할인, 쿠폰, 배송, 상품평, 구매 반응, 판매자, 쿠팡상품번호, 제품 특징/혜택 후보를 최대한 가져와 글 생성에 반영합니다.

단, 쿠팡 상세페이지의 제품 설명이 이미지로만 되어 있으면 이미지 속 글자는 자동으로 읽지 못할 수 있습니다. 그럴 때는 중요한 제품 설명을 `제품 스펙/구성/용량` 또는 `특징 키워드` 칸에 직접 붙여넣으면 더 정확하게 작성됩니다.


## 블로그 템플릿 저장/링크 분석 기능

이번 버전에는 `0. 블로그 템플릿` 영역이 추가되었습니다. 직접 만든 글 구조를 여러 개 저장해서 불러올 수 있고, 참고하고 싶은 블로그 링크를 여러 개 넣으면 문단 순서, 소제목 흐름, 도입부/마무리 방식, 사진/영상 배치 방식만 분석해서 새 글에 반영합니다. 원문 문장은 그대로 복사하지 않도록 프롬프트에 안전 규칙을 넣었습니다. 자세한 내용은 `블로그_템플릿_사용법.md`를 확인하세요.
