# 네이버 블로그 자동 작성/미리보기 도우미 - PC버전 + 핸드폰버전

이번 버전은 단순히 프롬프트를 보내는 프로그램이 아니라, 블로그 글을 생성하고 실제 네이버 블로그처럼 미리보기한 뒤 작성까지 이어지도록 만든 버전입니다.

---

## 포함된 버전

### 1. PC버전

폴더: `PC버전`

- Windows PC에서 실행합니다.
- `실행_PC용.bat`을 더블클릭해서 사용합니다.
- 입력칸을 전부 채우지 않아도 AI가 글을 생성합니다.
- 사진과 동영상을 업로드할 수 있습니다.
- 네이버 블로그처럼 보이는 미리보기를 제공합니다.
- 마음에 들면 Chrome을 열어 네이버 블로그 글쓰기 화면에 제목과 본문을 자동 입력합니다.
- `build_exe.bat`으로 EXE를 만들 수 있습니다.

중요: 네이버 로그인과 최종 발행은 직접 해야 합니다. 자동 발행은 넣지 않았습니다.

---

### 2. 핸드폰버전 Streamlit Cloud

폴더: `핸드폰버전_StreamlitCloud`

- PC를 켜두지 않아도 핸드폰에서 주소로 접속해 사용할 수 있는 방식입니다.
- GitHub + Streamlit Cloud에 올려서 사용합니다.
- 사진/동영상 업로드, AI 글 생성, 네이버식 미리보기, 제목/본문 복사가 가능합니다.
- API 키는 Streamlit Cloud의 Secrets에 넣습니다.

핸드폰 단독 버전은 보안상 네이버 편집기를 자동 조작하지 않습니다. 제목/본문을 복사하고 네이버 글쓰기 화면에 붙여넣는 방식입니다.

---

### 3. APK WebView 방식

폴더: `핸드폰버전_APK_WebView`

- Streamlit Cloud 주소를 Android 앱 안에서 열어주는 APK 프로젝트입니다.
- Android Studio에서 열고 APK로 빌드합니다.
- 완전 오프라인 앱이 아니라, Streamlit Cloud 주소를 앱처럼 열어주는 방식입니다.

## API 키 넣는 위치

### PC버전

`PC버전/.env` 파일을 메모장으로 열고 아래 줄을 바꿉니다.

```txt
NVIDIA_API_KEY=YOUR_API_KEY
```

`YOUR_API_KEY` 부분만 실제 NVIDIA API 키로 바꾸세요.

### 핸드폰버전 Streamlit Cloud

Streamlit Cloud의 Secrets에 아래 내용을 넣습니다.

```toml
NVIDIA_API_KEY = "YOUR_API_KEY"
NVIDIA_BASE_URL = "https://integrate.api.nvidia.com/v1"
NVIDIA_MODEL = "minimaxai/minimax-m3"
```

요청하신 대로 기본 base_url은 `https://integrate.api.nvidia.com/v1`로 바꿔두었습니다.
다만 프로그램은 실제 API 호출 시 오류 방지를 위해 OpenAI 호환 API 주소로 자동 보정할 수 있게 만들었습니다.

## 추천 사용 방식

PC에서 네이버 글쓰기까지 최대한 자동화하려면:

```txt
PC버전 실행
→ 정보 입력
→ 생성하기
→ 네이버식 미리보기 확인
→ 수정/작성하기
→ 미리보기 내용으로 네이버 블로그에 작성하기
→ 네이버 화면에서 최종 확인 후 발행
```

핸드폰 단독으로 쓰려면:

```txt
Streamlit Cloud 배포
→ 핸드폰 접속
→ 생성/미리보기
→ 제목/본문 복사
→ 네이버 블로그 앱 또는 모바일 웹에 붙여넣기
```

## NVIDIA 주소 관련 주의

`.env`의 `NVIDIA_BASE_URL`은 아래 값이 맞습니다.

```txt
https://integrate.api.nvidia.com/v1
```

이 주소는 브라우저로 여는 홈페이지가 아니라 API 기본 주소입니다.
주소창에 직접 입력하면 404가 뜰 수 있습니다. 프로그램은 내부에서 `/chat/completions`를 붙여 POST 요청을 보냅니다.


## NVIDIA API 타임아웃이 날 때

`Read timed out` 오류는 API 키 오류가 아니라 NVIDIA 서버 응답이 오래 걸려서 끊긴 상태입니다.
이번 버전은 기본 `NVIDIA_MAX_TOKENS=2500`, `NVIDIA_TIMEOUT_SECONDS=240`으로 조정했습니다.
그래도 느리면 왼쪽 설정에서 **AI 응답 길이 제한**을 1500~2500으로 낮춘 뒤 다시 생성하세요.

## 이번 최신 수정: 링크 글 + 리뷰어 후기 + 제품 상세 이미지 분석

이번 버전에서는 아래 기능을 추가했습니다.

- 제품/장소 링크 여러 개를 읽어 정보 수집
- 다른 리뷰어 블로그/후기 링크 여러 개를 참고
- 링크 페이지 안의 대표/상세 이미지 URL 후보 추출
- 사용자가 올린 제품 상세 설명 이미지/캡처 이미지 분석
- 상세 이미지 안에 적힌 제품 특징, 혜택, 사용법, 주의사항을 블로그 본문에 반영
- AI 결과에 `제목:`, `본문:`, `해시태그:` 라벨이 섞여도 미리보기/네이버 작성 전에 자동 제거

자세한 사용법은 `상세이미지_리뷰분석_사용법.md` 파일을 확인하세요.


## 블로그 템플릿 저장/링크 분석 기능

이번 버전에는 `0. 블로그 템플릿` 영역이 추가되었습니다. 직접 만든 글 구조를 여러 개 저장해서 불러올 수 있고, 참고하고 싶은 블로그 링크를 여러 개 넣으면 문단 순서, 소제목 흐름, 도입부/마무리 방식, 사진/영상 배치 방식만 분석해서 새 글에 반영합니다. 원문 문장은 그대로 복사하지 않도록 프롬프트에 안전 규칙을 넣었습니다. 자세한 내용은 `블로그_템플릿_사용법.md`를 확인하세요.
