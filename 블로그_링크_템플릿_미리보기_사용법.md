# 블로그 링크 템플릿 미리보기 사용법

이번 버전은 템플릿을 먼저 선택해서 쓰는 방식이 아니라, 마음에 드는 블로그 링크를 넣으면 그 글의 구조를 분석해서 템플릿 미리보기를 먼저 보여줍니다.

## 사용 순서

1. 앱을 실행합니다.
2. `0. 블로그 템플릿` 영역으로 갑니다.
3. `구조를 분석할 블로그 링크 여러 개` 칸에 마음에 드는 블로그 링크를 한 줄에 하나씩 넣습니다.
4. `링크 분석해서 템플릿 미리보기` 버튼을 누릅니다.
5. 아래에 나온 템플릿 미리보기를 확인합니다.
6. 마음에 들면 `이 분석 결과를 저장할 템플릿 이름`을 입력합니다.
7. `미리보기 템플릿 저장`을 누릅니다.
8. 이후 글 생성 때 저장한 템플릿을 불러오거나, `이번 글 생성에 이 분석 템플릿 바로 적용`을 켜고 바로 생성합니다.

## 분석하는 항목

- 제목 규칙
- 도입부 흐름
- 본문 구조와 소제목 순서
- 사진/동영상 배치 흐름
- 마무리 방식
- 해시태그 규칙
- 피해야 할 표현

## 중요한 점

다른 블로그 문장을 그대로 복사하지 않습니다. 구조와 흐름만 참고하고, 내용은 사용자가 입력한 제품/장소 정보와 업로드한 사진, 상세 이미지, 참고 링크 자료를 바탕으로 새로 작성합니다.

## 링크가 안 읽히는 경우

네이버 블로그나 일부 사이트는 외부 접근을 막을 수 있습니다. 이 경우 읽힌 링크만 반영하고, 안 읽힌 링크는 미리보기의 분석 결과에서 확인할 수 있습니다.
