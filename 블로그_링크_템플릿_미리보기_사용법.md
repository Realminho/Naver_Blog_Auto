# 블로그 링크 템플릿 미리보기 사용법

이번 버전에서는 블로그 링크를 넣으면 바로 저장하지 않고, 먼저 네이버 블로그 화면처럼 보이는 **양식 미리보기**를 보여줍니다.

## 사용 순서

1. `0. 블로그 템플릿` 영역으로 이동합니다.
2. `구조를 분석할 블로그 링크 여러 개` 칸에 참고하고 싶은 블로그 링크를 한 줄에 하나씩 넣습니다.
3. `링크 분석해서 템플릿 미리보기` 버튼을 누릅니다.
4. 화면에 `제목 자리`, `부제목 자리`, `본문 문단 자리`, `사진 자리`가 보이는 네이버식 템플릿 미리보기가 나타납니다.
5. 마음에 들면 템플릿 이름을 입력하고 `미리보기 템플릿 저장`을 누릅니다.
6. 저장한 템플릿은 이후 글 생성 때 불러와 사용할 수 있습니다.

## 중요한 점

- 참고 블로그의 제목이나 소제목을 그대로 복사하지 않습니다.
- `댕맘일기`, `자세히 보기`, `레이어 닫기` 같은 블로그 UI 문구가 템플릿에 들어가지 않도록 일반화합니다.
- 미리보기에서는 실제 제목 대신 `제목 자리`, 실제 소제목 대신 `부제목 1`, `부제목 2`처럼 양식으로 보여줍니다.
- 실제 블로그 글을 생성할 때는 사용자가 입력한 제품/장소 정보와 참고 링크 분석 내용을 바탕으로 자동으로 채워집니다.
