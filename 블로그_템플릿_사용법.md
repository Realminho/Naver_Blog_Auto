# 블로그 템플릿 저장/링크 분석 사용법

이번 버전에는 블로그 템플릿 기능이 추가되었습니다.

## 1. 내가 직접 템플릿 저장하기

프로그램 상단의 `0. 블로그 템플릿` 영역에서 아래 내용을 입력할 수 있습니다.

- 제목 규칙
- 도입부 규칙
- 본문 구조/소제목 순서
- 사진/동영상 배치 규칙
- 마무리 규칙
- 해시태그 규칙
- 피할 표현
- 템플릿 메모/고정 문구

입력 후 `현재 템플릿 저장`을 누르면 PC에 저장됩니다. 다음에 프로그램을 다시 열어도 `저장된 템플릿 불러오기`에서 다시 선택할 수 있습니다.

## 2. 블로그 링크를 템플릿처럼 참고하기

`구조를 참고할 블로그 링크 여러 개` 칸에 참고하고 싶은 블로그 주소를 한 줄에 하나씩 넣습니다.

예시:

```txt
https://blog.naver.com/...
https://m.blog.naver.com/...
```

이 링크는 제품 정보가 아니라 글 구조 참고용입니다. 프로그램은 링크의 원문을 그대로 복사하지 않고 아래 요소만 분석해서 참고합니다.

- 도입부 방식
- 소제목 순서
- 문단 길이
- 리스트/체크박스 사용 여부
- 사진/영상 배치 흐름
- 마무리 방식
- 해시태그 스타일

## 3. 제품 정보 링크와 템플릿 링크의 차이

`리뷰할 제품/장소 정보 링크 여러 개`는 제품 특징, 가격, 혜택, 장소 정보 등을 풍부하게 만들기 위한 칸입니다.

`참고할 다른 블로그/리뷰 링크 여러 개`는 다른 리뷰어가 느낀 장점, 아쉬운 점, 방문 팁 등을 참고하기 위한 칸입니다.

`구조를 참고할 블로그 링크 여러 개`는 글 구조와 말투만 참고하기 위한 칸입니다.

## 4. 추천 사용법

가장 좋은 흐름은 아래와 같습니다.

```txt
1. 제품/장소 링크 입력
2. 다른 리뷰어 후기 링크 입력
3. 구조를 따라가고 싶은 블로그 링크 입력
4. 내가 직접 느낀 장점/아쉬운 점 입력
5. 생성하기
6. 미리보기 확인 후 수정
7. 네이버 블로그에 작성하기
```

## 5. 주의사항

네이버 블로그, 쿠팡, 일부 쇼핑몰은 자동 읽기를 막는 경우가 있습니다. 이 경우 읽힌 정보만 반영되며, 꼭 넣어야 하는 제품 설명은 직접 입력하거나 상세페이지 캡처 이미지를 업로드하는 것이 가장 정확합니다.
